#### **Activity 10: Build code to find the gems (easy version)**

**The Activity:** Boost your brain's logic processing with games specifically designed to build programming logic. Using a block based programming system, control your character to collect the gems.

**Why it's fun:** These are genuinely engaging puzzle games that get progressively more challenging. They directly translate to the skills you'll need for writing real code, building your problem-solving muscles in an enjoyable and rewarding way that doesn't need any knowledge of programming.

**How to do it:**

1. Visit the [CodeCombat website]([https://codecombat.com/play/junior](https://codecombat.com/play/junior)) and start collecting.
